模板语法说明

基础类型

基础变量：{{ name }}

列表list

列表： {{ data }}
可以根据下表去直接取：{{ data.0 }}
列表的循环展示
{% for item in data %}
{{ item }}
{% endfor %}

字典dict

字典：{{ dictData }}
直接取属性：{{ dictData.name }}
字典的循环展示
注意这里的.items是字典的方法，所以要使用.items {% for k,v in dictData.items %}
key值：{{ k }}, value值: {{ v }}
{% endfor %}

if语句

{% if name == 'zhangsan' %}
zhangsan
{% else %}
{{ name }}
{% endif %}
